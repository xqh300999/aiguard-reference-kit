# 入门指南

> **用途**：本文档是入门模块的目录说明，介绍入门阶段需要了解的内容和阅读顺序。

## 模块概述

入门指南模块旨在帮助开发者快速搭建开发环境、了解项目全貌、选择适合自己的学习路径。无论你是后端、前端、嵌入式还是全栈开发者，都能在这里找到适合自己的起点。

## 内容列表

| 文档 | 用途 | 适用人群 |
|------|------|---------|
| [prerequisites.md](prerequisites.md) | 开发环境依赖说明与安装指南 | 所有开发者 |
| [learning-path.md](learning-path.md) | 分角色的学习路径规划 | 所有开发者 |
| [dev-basics.md](dev-basics.md) | Git/Docker/Screen/命令行等基础工具使用指南 | 初学者或需要复习的开发者 |

## 建议阅读顺序

1. **第一步**：阅读 [prerequisites.md](prerequisites.md)，检查并搭建本地开发环境
2. **第二步**：阅读 [learning-path.md](learning-path.md)，根据你的角色选择对应的学习路径
3. **第三步**：如对基础工具不熟悉，阅读 [dev-basics.md](dev-basics.md) 掌握必要的开发工具
4. **第四步**：进入 [01-agile-development](../01-agile-development/) 了解项目开发流程

## 前置知识

本资料包假设你具备以下基础知识（如有不足可通过 [dev-basics.md](dev-basics.md) 补充）：

- 基本的命令行操作能力
- 至少一种编程语言的基础
- 对 Web 或嵌入式开发有基本概念

> **提示**：不必等所有工具都安装完美再开始，边做边学也是很好的方式。遇到环境问题时可以随时查阅本文档或向团队求助。
