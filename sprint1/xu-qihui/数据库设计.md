# 数据库设计

## 数据库
- **名称**: `elderly_care`
- **字符集**: `utf8mb4`

---

## 表结构

### community（社区）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | bigint | PK, AUTO_INCREMENT | 社区 ID |
| name | varchar(100) | NOT NULL | 社区名称 |
| address | varchar(255) | | 地址 |
| area | varchar(100) | | 所属区域 |
| created_at | datetime | DEFAULT CURRENT_TIMESTAMP | 创建时间 |

---

### user（用户/账号）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | bigint | PK, AUTO_INCREMENT | 用户 ID |
| username | varchar(50) | NOT NULL, UNIQUE | 登录账号 |
| password | varchar(255) | NOT NULL | BCrypt 加密后的密码 |
| phone | varchar(20) | | 手机号 |
| role | enum('ADMIN','COMMUNITY_WORKER','FAMILY') | NOT NULL | 角色 |
| real_name | varchar(50) | | 真实姓名 |
| community_id | bigint | FK → community.id | 所属社区 |
| elderly_id | bigint | FK → elderly.id | 绑定的老人（FAMILY 用） |
| status | enum('ENABLED','DISABLED') | DEFAULT 'ENABLED' | 账号状态 |
| created_at | datetime | DEFAULT CURRENT_TIMESTAMP | 创建时间 |

**说明：**
- `elderly_id`：家属(FAMILY)账号通过此字段绑定到具体的老人
- `role = FAMILY` 的用户此字段应为老人 id；其他角色为 null

---

### elderly（老人）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | bigint | PK, AUTO_INCREMENT | 老人 ID |
| name | varchar(50) | NOT NULL | 老人姓名 |
| age | int | | 年龄 |
| gender | enum('MALE','FEMALE') | | 性别 |
| address | varchar(255) | | 家庭住址 |
| phone | varchar(20) | | 电话 |
| emergency_contact | varchar(100) | | 紧急联系人 |
| health_notes | text | | 健康状况备注 |
| community_id | bigint | FK → community.id | 所属社区 |
| device_mac | varchar(50) | | 绑定的设备 MAC 地址 |
| status | enum('NORMAL','ALERT','HOSPITALIZED') | DEFAULT 'NORMAL' | 老人状态 |
| created_at | datetime | DEFAULT CURRENT_TIMESTAMP | 创建时间 |

**说明：**
- `device_mac`：关联到 device 表的 mac 字段
- `status = ALERT`：有未处理的告警；`HOSPITALIZED`：已住院

---

### device（设备）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | bigint | PK, AUTO_INCREMENT | 设备 ID |
| name | varchar(100) | | 设备名称 |
| type | enum('WATCH','SMOKE','GAS','DOOR_MAGNET','TEMP_HUMID','PIR','CAMERA','DOORBELL','SMART_PLUG','SMART_SWITCH') | | 设备类型 |
| mac | varchar(50) | UNIQUE | MAC 地址 |
| community_id | bigint | FK → community.id | 所属社区 |
| elderly_id | bigint | FK → elderly.id | 绑定的老人 |
| status | enum('ONLINE','OFFLINE','FAULT') | DEFAULT 'OFFLINE' | 设备状态 |
| battery | int | DEFAULT 0 | 电量百分比 |
| last_heartbeat | datetime | | 最后心跳时间 |
| created_at | datetime | DEFAULT CURRENT_TIMESTAMP | 创建时间 |

---

### alert（告警）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | bigint | PK, AUTO_INCREMENT | 告警 ID |
| elderly_id | bigint | FK → elderly.id | 老人 ID |
| community_id | bigint | FK → community.id | 社区 ID |
| type | enum('SOS','SMOKE','GAS','DOOR_ABNORMAL','INACTIVITY','FALL','LOW_BATTERY','DEVICE_OFFLINE','ABNORMAL') | | 告警类型 |
| status | enum('PENDING','PROCESSING','RESOLVED') | DEFAULT 'PENDING' | 处理状态 |
| cause | varchar(255) | DEFAULT '不明' | 原因 |
| details | text | | 详情描述 |
| happened_at | datetime | | 发生时间 |
| resolved_at | datetime | | 解决时间 |
| handler_id | bigint | FK → user.id | 处理人 |
| created_at | datetime | DEFAULT CURRENT_TIMESTAMP | 创建时间 |

---

### dispatch（派单）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | bigint | PK, AUTO_INCREMENT | 派单 ID |
| alert_id | bigint | FK → alert.id | 关联告警 |
| handler_id | bigint | FK → user.id | 处理人 |
| status | enum('DISPATCHED','ARRIVED','COMPLETED') | DEFAULT 'DISPATCHED' | 派单状态 |
| description | text | | 描述 |
| result | enum('RESOLVED','NEED_HOSPITAL','NEED_FOLLOW_UP') | | 处理结果 |
| handled_at | datetime | | 处理时间 |
| created_at | datetime | DEFAULT CURRENT_TIMESTAMP | 创建时间 |

---

### care_plan（关怀计划）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | bigint | PK, AUTO_INCREMENT | 计划 ID |
| elderly_id | bigint | FK → elderly.id | 老人 ID |
| worker_id | bigint | FK → user.id | 负责工作人员 |
| type | enum('PHONE','VISIT') | | 类型：电话/上门 |
| frequency | varchar(100) | | 频率 |
| status | enum('ACTIVE','PAUSED') | DEFAULT 'ACTIVE' | 状态 |
| created_at | datetime | DEFAULT CURRENT_TIMESTAMP | 创建时间 |

---

### care_log（关怀记录）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | bigint | PK, AUTO_INCREMENT | 记录 ID |
| plan_id | bigint | FK → care_plan.id | 关联计划 |
| worker_id | bigint | FK → user.id | 工作人员 |
| type | enum('PHONE','VISIT') | | 类型 |
| status_change | text | | 状态变化 |
| notes | text | | 备注 |
| executed_at | datetime | | 执行时间 |
| created_at | datetime | DEFAULT CURRENT_TIMESTAMP | 创建时间 |

---

### community_stats（社区统计）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | bigint | PK, AUTO_INCREMENT | 统计 ID |
| community_id | bigint | FK → community.id | 社区 ID |
| period | enum('DAILY','WEEKLY','MONTHLY') | | 统计周期 |
| period_start | date | | 周期开始 |
| period_end | date | | 周期结束 |
| elderly_count | int | DEFAULT 0 | 老人数量 |
| device_count | int | DEFAULT 0 | 设备数量 |
| device_online | int | DEFAULT 0 | 在线设备数 |
| new_elderly_count | int | DEFAULT 0 | 新增老人数 |
| alert_total | int | DEFAULT 0 | 告警总数 |
| alert_resolved | int | DEFAULT 0 | 已解决告警数 |
| avg_response_minutes | int | DEFAULT 0 | 平均响应时间(分钟) |
| care_planned | int | DEFAULT 0 | 计划关怀数 |
| care_completed | int | DEFAULT 0 | 已完成关怀数 |
| created_at | datetime | DEFAULT CURRENT_TIMESTAMP | 创建时间 |

---

## 关系图

```
community ──┬── user
            ├── elderly ──┬── device
            │             ├── alert
            │             └── care_plan ── care_log
            ├── device
            └── community_stats

alert ── dispatch
user  ──┬── alert (handler)
        └── care_plan (worker)
```
