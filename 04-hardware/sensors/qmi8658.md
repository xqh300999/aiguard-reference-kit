# QMI8658 六轴IMU传感器

## 概述

QMI8658 是 QST（矽睿科技）推出的高性能六轴惯性测量单元（IMU），集成三轴加速度计和三轴陀螺仪。在 AIguard 参考套件中，QMI8658 板载于 ESP32-S3-Touch-AMOLED-2.06 手表开发板，用于运动姿态检测、跌倒检测、计步等功能。

## 技术规格

| 参数 | 加速度计 | 陀螺仪 |
|------|----------|--------|
| 轴数 | 3轴 (X/Y/Z) | 3轴 (X/Y/Z) |
| 量程 | ±2/±4/±8/±16g | ±16/±32/±64/±128/±256/±512/±1024/±2048dps |
| 分辨率 | 16位 | 16位 |
| 工作电流 | — | — |
| 低功耗模式 | 支持 | 支持 |
| 通信接口 | I2C/SPI | I2C/SPI |
| 工作电压 | 1.71V ~ 3.6V | 1.71V ~ 3.6V |

### I2C 特性
- I2C 地址：0x6B（默认，取决于SA0引脚电平）
- 支持最高 400kHz（Fast Mode）和 1MHz（Fast Mode Plus）
- 在 ESP32-S3-Watch 上，与触控芯片(FT3168)、RTC(PCF85063)共用I2C总线

## 板载连接

在 ESP32-S3-Touch-AMOLED-2.06 上的连接：

| QMI8658 引脚 | ESP32-S3 引脚 | 说明 |
|--------------|---------------|------|
| SCL | GPIO15 | I2C 时钟 |
| SDA | GPIO14 | I2C 数据 |
| INT1 | GPIO12 | 中断输出1 |
| SA0 | — | I2C地址选择 |

## 主要功能

### 1. 加速度计
- 检测线性加速度
- 静止/运动状态判断
- 倾角检测
- 自由落体检测

### 2. 陀螺仪
- 检测角速度
- 旋转方向判断
- 姿态角度计算

### 3. 运动检测
- 计步器
- 活动/静止识别
- 运动方向检测

### 4. 跌倒检测（AIguard重点功能）

跌倒检测实现思路：
1. **加速度阈值检测**：跌倒瞬间加速度会出现剧烈变化，合成加速度可能远超 2g
2. **姿态变化检测**：跌倒后人体姿态从直立变为躺卧，倾角发生明显变化
3. **撞击后静止检测**：跌倒撞击后通常伴随短暂静止
4. **多条件综合判断**：
   - 加速度峰值超过阈值（如 > 3.5g）
   - 姿态角度变化超过阈值（如 > 60°）
   - 撞击后一段时间内（如 2-5秒）加速度接近静止（1g附近波动）
5. **误报过滤**：
   - 快速坐下、弯腰等动作加速度变化模式不同
   - 结合持续时间、峰值形态综合判断
   - 可通过机器学习模型优化准确率

## 驱动配置

### Arduino 环境
使用 SensorLib 库（v0.3.1或更高版本）：

```cpp
#include <QMI8658.h>

QMI8658 qmi8658;

void setup() {
    Wire.begin(14, 15);  // SDA, SCL
    qmi8658.init();
    qmi8658.configAccelerometer(ACC_RANGE_8G, ACC_ODR_1000Hz);
    qmi8658.configGyroscope(GYRO_RANGE_256DPS, GYRO_ODR_1000Hz);
}
```

### ESP-IDF 环境
使用 QMI8658 组件驱动，通过 I2C 接口读写寄存器。

### 推荐采样配置
- 加速度计量程：±8g
- 陀螺仪量程：±256dps（日常活动）或 ±512dps（跌倒检测）
- 输出数据率 (ODR)：100Hz ~ 500Hz（跌倒检测建议 ≥ 200Hz）
- 启用低通滤波减少噪声

## 数据处理

### 坐标系
QMI8658 通常使用右手坐标系，具体方向取决于芯片在PCB上的安装方向。需根据实际安装位置进行坐标变换。

### 常用算法
- **互补滤波**：简单姿态解算，资源占用低
- **Mahony/Madgwick 滤波**：较准确的姿态解算，适合MCU
- **卡尔曼滤波**：更高精度但计算量较大
- **简单峰值检测**：用于跌倒检测的实时判断

## 低功耗配置

为延长手表电池续航：
- 正常监测模式：ODR 50Hz，加速度计低功耗模式
- 运动触发：当加速度变化超过阈值时切换到高频率采样
- 睡眠模式：长时间静止时降低 ODR 或进入休眠
