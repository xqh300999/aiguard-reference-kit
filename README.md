# AiGuard 参考资料包

> **用途**：本文档是 AiGuard 参考资料包的入口说明，介绍资料包定位、内容结构、知识地图与使用方式。

## 资料包定位

AiGuard 参考资料包是一套面向智能家居养老看护系统开发的学习与实践指南。本资料包：

- ✅ 提供功能目标与需求说明
- ✅ 提供参考架构与最佳实践
- ✅ 提供开发流程与敏捷实践模板
- ❌ 不限制具体技术实现方式
- ❌ 不强制使用特定框架或工具
- 💡 鼓励创新与探索更优方案

## 阅读建议

### 按角色阅读

| 角色 | 推荐阅读顺序 |
|------|-------------|
| 后端开发 | 00 → 01 → 02 → 技术栈选择 → 后端实现 |
| 前端开发 | 00 → 01 → 02 → 技术栈选择 → 前端/面板实现 |
| 嵌入式开发 | 00 → 01 → 02 → 硬件平台 → 固件开发 |
| 全栈开发 | 00 → 01 → 02 → 按需选择各模块深入 |
| 项目经理/Scrum Master | 00 → 01 → 02 → 重点关注敏捷实践部分 |

### 阅读阶段

1. **第一阶段（1-2天）**：通读 00-getting-started，搭建开发环境，了解学习路径
2. **第二阶段（2-3天）**：学习 01-agile-development，理解敏捷流程与 Sprint 节奏
3. **第三阶段（3-5天）**：深入 02-project-overview，理解功能需求与架构目标
4. **第四阶段（持续）**：根据所选角色，进入具体技术实现阶段

## Skill 安装说明

本资料包可配合 AI 编程助手的 Skill 机制使用，提升开发效率。

### 什么是 Skill？

Skill 是 AI 编程助手的扩展能力模块，可以为特定领域或项目提供专业的知识、模板和工作流。

### 安装方式

1. 确认你使用的 AI 编程助手支持 Skill 功能（如 TRAE 等）
2. 在助手的 Skill 管理界面中，选择「从本地目录安装」或「导入 Skill」
3. 选择本资料包中对应的 Skill 目录（如有）
4. 按照提示完成安装
5. 安装完成后，即可在对话中调用相关 Skill 获得专业辅助

### 推荐使用场景

- Sprint 规划会议：使用敏捷相关 Skill 辅助生成会议纪要与任务拆分
- 代码开发：使用技术栈相关 Skill 获取代码模板与最佳实践
- 问题排查：使用调试相关 Skill 辅助定位问题
- 文档编写：使用文档相关 Skill 辅助生成规范文档

> **提示**：Skill 是辅助工具，最终决策与实现仍由开发者主导。鼓励根据实际项目需求自定义 Skill。
